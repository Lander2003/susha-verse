# SushaVerse

SushaVerse is a full-stack Bible study application built with React, TypeScript, Node.js, Express, and PostgreSQL.

Users will be able to:

- Create an account and authenticate
- Browse books, chapters, and verses
- Save verses for later
- Create personal notes and reflections
- Track reading activity and streaks
- View a personalized dashboard

## Tech Stack

### Frontend
- React
- TypeScript

### Backend
- Node.js
- Express

### Database
- PostgreSQL
